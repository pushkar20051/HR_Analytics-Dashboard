# HR Analytics Dashboard

## Overview
This project presents an HR Analytics Dashboard developed using Tableau Public to analyze employee attrition, demographics, job satisfaction, and workforce trends.

## Objectives
- Analyze employee attrition patterns
- Understand workforce demographics
- Evaluate job satisfaction levels
- Support data-driven HR decision-making

## Tools Used
- Tableau Public
